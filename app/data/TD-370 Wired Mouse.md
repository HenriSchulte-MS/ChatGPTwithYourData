# Contoso TD-370 Wired Mouse User Guide

The Contoso TD-370 Wired Mouse is a comfortable and precise mouse that connects to your computer via a USB cable. It features a scroll wheel, two buttons, and a DPI switch that lets you adjust the sensitivity of the mouse. It also has a rubberized grip and a smooth glide.

## Getting started

To use the Contoso TD-370 Wired Mouse, follow these steps:

1. Plug the USB connector of the mouse into an available USB port on your computer.
2. Wait for the computer to recognize the mouse and install the necessary drivers. You may see a notification on your screen when this is done.
3. Start using your mouse. You can use the scroll wheel to scroll up and down web pages and documents. You can use the left button to click and select items, and the right button to access context menus. You can use the DPI switch to change the speed of the cursor movement.
4. To customize your mouse settings, such as the button functions and the pointer options, you can use the Contoso Mouse Software that comes with your mouse. You can download it from www.contoso.com/software.

## Troubleshooting

If you encounter any problems with your mouse, try these solutions:

- Make sure the USB cable is securely plugged into both the mouse and the computer.
- Try plugging the mouse into a different USB port or a different computer.
- Check if there are any updates for your mouse drivers or software. You can visit the Contoso website (www.contoso.com) for more information.
- Clean the mouse with a soft cloth and a mild cleaner. Do not immerse the mouse in water or use harsh chemicals.
- Contact Contoso customer support if none of the above solutions work.
